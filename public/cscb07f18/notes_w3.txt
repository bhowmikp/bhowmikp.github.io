What is SVN?
- Subversion (SVN) is a centralized version control system. It keeps track of
  files and directories and how changes are made to them over time. We can
  recover older version of our projects and see how the project has evolved
  over time.
- Subversion allows multiple people to work on the same project as it can be
  operated across networks.

What is Java?
- Java is an Object-Oriented programming language.
- Java source code is complied into bytecode
- Everything is a class in Java
